# Homebrew Tap for Hooklistener CLI

This is the official [Homebrew](https://brew.sh) tap for [Hooklistener CLI](https://github.com/hooklistener/hooklistener-cli).

## Installation

```bash
brew tap hooklistener/tap
brew install hooklistener
```

## Update

```bash
brew update
brew upgrade hooklistener
```

## Uninstall

```bash
brew uninstall hooklistener
brew untap hooklistener/tap
```

## About Hooklistener CLI

A fast, terminal-based CLI for browsing webhooks, forwarding events, and exposing local servers using [Hooklistener](https://hooklistener.com). Built with Rust and Ratatui for a smooth, responsive TUI experience.

For more information, visit the [main repository](https://github.com/hooklistener/hooklistener-cli).
